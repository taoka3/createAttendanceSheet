# 📋 自動勤怠管理システム - README

このシステムは、Google スプレッドシート上でボタンを押し、ポップアップ画面に「年」と「月」を入力するだけで、その月の日数や曜日に合わせた専用の勤怠管理表シートを1秒で自動生成するシステムです。

---

## 1. システム概要

### 💡 主な機能

* **スマート生成**: 月ごとの正確な日数（28日〜31日）や曜日を自動判定してシートを作ります。
* **かんたん時間計算**: 開始・終了・休憩時間を入力するだけで、労働時間を自動計算。
* **カスタム残業設定**: **「1日7時間」** を超えた労働を自動的に「残業時間」として別カウントします。
* **カラフルなプルダウン**: 勤務タイプを選ぶと、その内容に合わせてセルが自動で見やすく色分けされます。
* **ゆとりあるデザイン**: 文字が詰まって見づらくならないよう、セルの幅を自動で最適化（余白付き）します。

### 🎨 勤務タイプと色分けルール

| 勤務タイプ | セルの背景色 | 想定される用途 |
| --- | --- | --- |
| **通常出勤** | 通常（白） | オフィス等への通常出勤時 |
| **リモートワーク** | 🟢 薄い緑 | 在宅勤務やサテライトオフィス勤務時 |
| **有給休暇 / 休暇 / お休み** | 🔴 薄い赤 | 公休、公傷、有給取得日など終日休み |
| **有給半休 / 半休** | 🟡 薄い黄 | 午前休・午後休などの半日勤務時 |

## 2. 設置・設定方法

初めて導入する際は、以下のステップに沿って3分ほどで設定が完了します。

1. **GASコードの貼り付け:** 所要時間: 1分.
1. 使用したいGoogle スプレッドシートを開きます。
2. メニューの **「拡張機能」 ＞ 「Apps Script」** をクリックします。
3. 最初から入っているコード（`myFunction`など）をすべて消去します。
4. 提供された最新のGASコードを丸ごと貼り付け、**保存アイコン（フロッピーマーク）** を押して保存します。


2. **起動ボタンの作成:** 所要時間: 1分.
1. スプレッドシートの最初のシートに戻ります。
2. メニューの **「挿入」 ＞ 「図形描画」** をクリックします。
3. 工具箱のようなマークから「角丸長方形」などの図形を選んで画面に描き、文字で「**勤怠管理作成**」と入力して右上の「保存して閉じる」を押します。
4. シート上に表示されたボタンをドラッグして、押しやすい位置に配置します。


3. **スクリプトの割り当て:** 所要時間: 30秒.
1. 配置したボタン（図形）を1回クリックして選択状態にします。
2. ボタンの右上に出てくる **「縦の三点リーダー（︙）」** をクリックします。
3. メニューから **「スクリプトを割り当て」** を選択します。
4. 入力窓に `createAttendanceSheet` と正確に入力し、「OK」を押します。

## 3. 使い方と注意点

### 🚀 実際の動かし方

1. 作成した「勤怠管理作成」ボタンをクリックします。
2. 画面にポップアップ（入力欄）が出るので、**「年（例: 2026）」** ➔ **「月（例: 7）」** の順に半角数字で入力して「OK」を押します。
3. 指定した年月の新しいシートが瞬時に作成されます。

> ⚠️ **【重要】初回実行時の注意（アクセスの承認）**
> 初めてボタンを押したときだけ、「承認が必要です」というGoogleのセキュリティ画面が出ます。これは自作のプログラムを動かすために必要な正常なステップです。
> **「続行」 ➔ 「自分のアカウントを選択」 ➔ 「詳細」 ➔ 「（安全ではないページ）に移動」 ➔ 「許可」** の順にクリックして進めてください。

### ✍️ 日常の入力ルール

* **時間入力のコツ**:
時間は必ず `9:00` や `17:30`、`1:00` のように**半角のコロン区切り**で入力してください。全角で入力すると計算エラー（`#VALUE!`）になります。
* **残業の計算**:
1日の労働時間が「7時間」を超えると自動で計算されます。もし基準を8時間等に変えたい場合は、GASコード内の `"7:00"` の部分を書き換えることで簡単にカスタマイズ可能です。
